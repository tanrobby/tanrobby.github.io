Exercise 4: Divide and Conquer Algorithms

Exercise 4.1: Divide and Conquer Algorithms

  1. Binary Search:
    1. Given an input of a sorted array, write pseudocode to search a given element in the array using a divide and conquer algorithm. A simple linear search will take O(n), but using divide-and-conquer, also known as binary search, should take O(log n).
    2. Write code in python based on your pseudocode.
    3. Calculate the runtime of the recursive version of your binary search.
  2. Mergesort:
    1. Write code in python for the mergesort algorithm as described in the textbook. Modify the code, so that you can see every split arrays, and every merged arrays.
    2. Show in detail that your implemented mergesort is O(n log n).
  3. Recursive insertion sort:
    1. Write pseudocode for recursive insertion sort.
    2. Write code in python based on your pseudocode.
    3. Calculate the runtime of the recursive version of your insertion sort in big-O.
  4. Strassen's algorithm for matrix multiplication:
    1. Write pseudocode in python to implement Strassen's algorithm described in the textbook Sec.4.2 (page 75).
    2. Write code in python based on your pseudocode.
    3. Calculate the runtime of your Strassen code.
