Exercise 1: Python Programming and Basic Concepts of Runtime Analysis

Exercise 1.1: Search and Swap Functions

  1. Write a function to swap an element with another element in the same list based on their index.
  2. Write a function to search an integer from an array (or list) of integers.
  3. Write a function to search a character from a list of characters.
  4. Write a function that accepts a list of integers and a number, and outputs the division of the list by the number.

Exercise 1.2: Argument Passing in Python

  1. If we want to pass a variable to a function without expecting the variable to change its value after executing the function (i.e., passing by value), how can we do it? Provide an example.
  2. If we want to pass a variable to a function and expect some change of the value of the variable (i.e., passing by reference), how can we do it? Provide an example.
  3. In Python, argument passing for functions is based on "passing by object reference". What does it mean by this? Discuss how its work and provide an example.

Exercise 1.3: For Loops, While Loops, Break and Continue

  1. If you use for loops in Exercise 1.1., change them to while loops.
  2. Find an example of a case that uses 'break', and write its python code.
  3. Find an example of a case that uses 'continue', and write its python code.

Exercise 1.4: Selection Sort

  1. Write a python code for Selection Sort based on its pseudocode.
  2. Change the code, so that the sorting is done in a descending order.

Exercise 1.5: Loop Runtime Analysis

  1. Given a 1D array, if we use a linear search algorithm, what is the runtime? Prove your answer by first writing the pseudocode, and then showing the runtime computation.
  2. The same question as number 1 above but for a 2D array.
  3. The same question as number 1 above but for a 3D array.
